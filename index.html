WASM

WebAssembly

WebAssembly es un conjunto de instrucciones binarias para una maquina virtual de pila. Fue diseñado para proveer un formato de codigo compilable y portable para otros lenguajes de programacion.

La maquina de pila de WASM esta diseñada para ser codificada en un formato binario eficinete en tamaño y tiempo de carga. Tiene como objetivo ejecutar a velocidades nativas, tomando ventaja de las capacidades del hardware disponibles en la mayoria de plataformas.

WebAssembly describe una esquema de "memoria segura", y ejecucion en "una caja de arena".
Esto significa que ese codigo no puede comunicarse directamente con el Sistema Operativo. El host necesita proveer un API para acceder a las fuciones del Sitema Operativo. El host puede limitar lo que la caja de arena puede realizar. En el mejor de los casos las exepciones son realizas explicitamente.

WASI

WebAssembly System Interface

Es una Interfaz de Sistema para WebAssembly, conformada por colecciones modulares de APIs estandarizadas enfocadas en Seguridad y Portabilidad. Estas APIs son principalmente para sistemas de archivos, network, graficos, audio y dispositivos de entrada
